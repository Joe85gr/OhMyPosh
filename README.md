# OhMyPosh_Install
This powershell script will install and configure "Oh My Posh" on Windows Terminal powershell.

## Requirements:
- Windows Terminal must be installed (this can be downloaded from the Microsoft Store)

## How To Run:
- Open Windows Terminal Powershell
- Navigate to the folder where you downloaded the application
- Run ``` .\install_oh_my_posh.ps1 ```
